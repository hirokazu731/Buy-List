# ホームシアター配置検討

## 前提（入力リンク由来）
- 椅子は「高級シアターチェア固定」ではなく、ニトリ等の一般家具も候補に含める。
- 映像側は4Kプロジェクター（XGIMI TITAN系）の画質志向。
- 室音響は防音・吸音をセットで計画する必要あり。

## 配置方針（初版）
1. **視聴ポジション先行で決める**
   - 椅子サイズとリクライニング時奥行きを確定。
   - 背面通路を確保し、出入りと掃除の動線を作る。
2. **投写距離を逆算してプロジェクター位置を決める**
   - スクリーンサイズ（例: 100〜120インチ）を仮置きし、投写距離を試算。
   - 天吊り/棚置きのどちらが騒音・配線・メンテで有利か比較。
3. **防音と吸音の優先順位を分離**
   - 近隣配慮が先なら遮音（壁/窓/ドア隙間）を優先。
   - 聴感改善が先なら一次反射面（側壁・天井）吸音を優先。

## 仮レイアウト案
- 前方：スクリーン + センター想定位置
- 中央：メインチェア1〜2脚（ニトリ候補含む）
- 後方：必要に応じてサラウンド/収納
- 天井または後方棚：4Kプロジェクター

## 未確定項目
- 部屋寸法（縦横高さ）
- 目標スクリーンサイズ
- スピーカーch構成（5.1 / 7.1 / 7.1.4）
- 家具搬入経路（ドア幅・廊下幅）
