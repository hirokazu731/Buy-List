# ホームシアター設計士確認リスト

## A. 椅子（ニトリ候補）
- [ ] 長時間視聴での腰・首サポート性能は十分か
- [ ] リクライニング時に後方クリアランスは足りるか
- [ ] 座面高さとスクリーン中心高さの整合は取れるか
- [ ] 床材との相性（滑り/床傷）対策は必要か

## B. プロジェクター（XGIMI TITAN候補）
- [ ] 現行流通しているか（新品/中古）
- [ ] 目標インチを満たす投写距離か
- [ ] 明るさ・黒表現が部屋環境に適合するか
- [ ] 動作音が視聴距離で許容範囲か
- [ ] ゲーム用途想定時の入力遅延は許容範囲か

## C. 共通
- [ ] 将来の機材更新を見据えた配線経路を確保しているか
- [ ] 電源容量（専用回路の必要性）を満たしているか
- [ ] 排熱・換気計画に問題がないか
