# ホームシアター追加調査リスト

## すぐ調べる（高優先）
- [ ] ニトリの現行リクライニングチェアで「ヘッドレスト高いモデル」を抽出
- [ ] XGIMI TITANの中古/後継機の流通状況と価格帯
- [ ] 100/110/120インチ時の投写距離早見表を作成
- [ ] 夜間視聴を想定した騒音許容値（dB）を定義

## 設計士確認前に準備
- [ ] 部屋実測（幅・奥行・天井高・ドア寸法）
- [ ] コンセント位置、分電盤回路、LAN配線可否
- [ ] 近隣への音漏れ懸念箇所（壁/窓/ドア）を写真で記録

## 比較検討（中優先）
- [ ] プロジェクター候補3機種で比較表（輝度/コントラスト/遅延/騒音）
- [ ] 椅子候補3モデルで比較表（座面高/全高/重量/価格）
- [ ] 吸音材候補（パネル/カーテン/ラグ）の費用対効果

## 後回しでよい（低優先）
- [ ] 間接照明の演出パターン
- [ ] 壁色・天井色の最終意匠
